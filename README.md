# Widgets Flutter — Colección de componentes reutilizables

Librería de widgets Flutter personalizados y reutilizables, desarrollados para ser compartidos entre múltiples aplicaciones del ecosistema de proyectos Flutter.

## Widgets incluidos

- **audio_player_controls** — Controles de reproducción de audio con play, pausa, avance y barra de progreso
- **buttons_edit_delete_save** — Botones de acción estándar para editar, eliminar y guardar
- **cards** — Tarjetas reutilizables para mostrar contenido con diferentes layouts
- **dashboard_current_status_cards** — Tarjetas de estado para dashboards con indicadores visuales
- **image_picker_card** — Tarjeta con selección de imagen desde galería o cámara
- **liquid_glass_nav_bar** — Barra de navegación inferior con efecto vidrio líquido
- **search_bar** — Barra de búsqueda con debounce y limpieza de texto
- **selectors** — Selectores de opciones: dropdown, chips, radio buttons y switches
- **sidebar_app_drawer** — Menú lateral (drawer) con navegación y perfil de usuario

## Tecnología

- Flutter (Android / iOS)
- Sin dependencias externas adicionales

## Cómo usar

Copia el widget deseado a tu proyecto y ajusta los parámetros según la interfaz pública de cada componente.

```bash
flutter pub get
flutter run
```
